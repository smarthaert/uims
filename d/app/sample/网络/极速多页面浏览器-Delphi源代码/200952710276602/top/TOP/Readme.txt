   本软件使用方便，内置有许多方便的快捷键可以使用。
   例如：
   CTRL+Y : 软件界面的显示/隐藏
   CTRL+K : 快速退出-不保存 (快速退出程序，但不保存所浏览内容等)
   此二组合键为全局热键，无论浏览器窗口是否为当前活动窗口皆有效。
   CTRL+B :  新建一空白页
   CTRL+T :  查看天气预报
   CTRL+W :  关闭当前活动页面
   CTRL+ALT+W : 关闭全部页面
   SHIFT+F5 : 刷新全部页面
   CTRL+Q : 查看前一页面
   CTRL+M : 查看后一页面
   CTRL+E :  退出程序
   以上组合键，当浏览器窗口为当前活动窗口时有效。
